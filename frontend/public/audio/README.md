# Highlight reel background music

Drop a single mp3 named **`highlights-music.mp3`** in this folder
(2-3 minutes, royalty-free, ideally upbeat). It will be auto-mixed
into highlight reels when the user toggles "Add background music".

Recommended sources (CC0/no-attribution):
- https://pixabay.com/music/ (filter: "Sport")
- https://uppbeat.io/free-music
- https://www.youtube.com/audiolibrary

If this file is missing, the music toggle has no effect — original
audio is used. Keep file under ~3 MB to avoid bloating cold loads.
